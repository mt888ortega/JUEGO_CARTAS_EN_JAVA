JUEGO DE CARTAS SIMPLE EN JAVA APLICANO POO.

EL GANADOR SERA EL PRIMER JUGADOR QUE CONSIGA 3 PAREJAS DE DISTINTO RANGO (SIN IMPORTAR EL PALO) EN SUS 6 CARTAS.

EJEMPLO:

        ✭TURNO DEL JUGADOR 2✭


1-CORAZONES ♥️ ➤ Rey
2-TREVOLES ♣️ ➤ Rey
3-CORAZONES ♥️ ➤ Reina
4-CORAZONES ♥️ ➤ As
5-DIAMANTES ♦️ ➤ As
6-DIAMANTES ♦️ ➤ Reina

        ::::::::::::::::::
        EL JUGADOR 2 GANÓ.
